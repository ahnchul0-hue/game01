# PRD: 카피바라 러너 (Capybara Runner)

## 1. 목표

온천을 찾아 떠나는 카피바라의 힐링 무한 러닝 게임.
"달리면서 힐링하는 유일한 온천 러너 게임"을 핵심 USP로,
가볍고 귀여운 감성으로 스트레스 해소와 수집의 재미를 동시에 제공한다.

## 2. 타겟 사용자

| 구분 | 설명 |
|------|------|
| 1차 타겟 | 10~30대, 캐주얼/힐링 게임 선호, 모바일 웹 사용자 |
| 2차 타겟 | 카피바라/동물 캐릭터 팬, SNS 공유형 유저 |
| 플랫폼 | 모바일 웹 (터치) + PC 웹 (키보드) |

## 3. 핵심 기능

### 3-1. 러닝 게임 코어
- 자동 달리기 + 3레인 좌우 이동
- 점프 / 슬라이드 장애물 회피
- 귤, 과일, 온천 재료 수집
- 거리 기반 점수 + 아이템 보너스

### 3-2. 파워업 시스템
- 수박 헬멧: 장애물 1회 방어
- 오리 튜브: 물 위 이동 (특수 구간)
- 친구 동물 합류: 일정 시간 무적 + 아이템 자동 수집

### 3-3. 온천 꾸미기 (메타 게임)
- 수집한 재료로 나만의 온천 건설/업그레이드
- 장식, 온천 종류, 배경 커스터마이징
- 동물 친구 배치

### 3-4. 스테이지 & 배경
- 숲속 → 강가 → 마을 → 온천 (4개 스테이지)
- 스테이지별 고유 장애물, 아이템, 배경음

### 3-5. 릴렉스 모드
- 장애물 감소 + 힐링 배경음 + 느긋한 속도
- 스트레스 없이 아이템 수집에 집중

### 3-6. 리더보드
- 최고 점수 랭킹
- 온천 자랑하기 (스크린샷 공유)

## 4. 경쟁사 분석

| 항목 | 토킹톰 골드런 | 카피바라 Go! | 카피바라 러너 (우리) |
|------|-------------|-------------|---------------------|
| 장르 | 무한 러너 | 로그라이크 RPG | 힐링 무한 러너 |
| 감성 | 액션/추격 | 전투/모험 | 힐링/귀여움 |
| 수집 | 금괴 → 집 수리 | 장비/캐릭터 | 온천 재료 → 온천 꾸미기 |
| 차별점 | 다양한 캐릭터 | 방치형 RPG | 온천 테마 + 릴렉스 모드 |
| 플랫폼 | 네이티브 앱 | 네이티브 앱 | 웹 (설치 불필요) |

### USP (Unique Selling Point)
- **온천 힐링**: 러너 장르에서 유일한 힐링 컨셉
- **웹 기반**: 설치 없이 URL 접속만으로 플레이
- **릴렉스 모드**: 경쟁이 아닌 힐링 목적의 게임 모드
- **동물 친구**: 카피바라의 사교적 특성 반영, 다양한 동물 동료

## 5. 에셋 전략

| 종류 | 방법 | 비고 |
|------|------|------|
| 캐릭터 스프라이트 | AI 생성 (이미지 생성 AI) + 수동 보정 | 카피바라, 동물 친구들 |
| 배경 | AI 생성 + 타일맵 조합 | 4개 스테이지별 |
| UI/아이콘 | 무료 에셋 + 자체 제작 | |
| 사운드/BGM | 무료 에셋 (자연/힐링 사운드) | freesound.org 등 |
| 이펙트 | Phaser 3 파티클 시스템 | 물 튀김, 귤 수집 등 |

## 6. 수익화 (향후)
- 1차: 무료 플레이 (광고 없음, 사용자 확보 우선)
- 2차: 선택적 광고 (부활 시 광고 시청)
- 3차: 프리미엄 스킨/온천 테마 (소액 결제)
